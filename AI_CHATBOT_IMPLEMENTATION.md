## 💡 How It Works - Example

### User asks: "Find running shoes"

**Step 1: Database Search**
```
Finding products with name/category matching "running shoes"
→ Found: Running Shoes (ID: 1, $120, Stock: 10)
```

**Step 2: AI Context**
```
System Prompt gets all products:
"Available Products:
- Running Shoes: $120, Stock: 10, Category: Shoes
- Basketball Shoes: $150, Stock: 8, Category: Shoes
- Tennis Racket: $80, Stock: 5, Category: Sports
..."
```

**Step 3: Groq AI Generates**
```
"Based on your search, I found Running Shoes which are perfect 
for outdoor training. They're priced at $120 with 10 in stock. 
Similar options include Basketball Shoes and other athletic footwear."
```

**Step 4: Recommendations**
```
Other shoes from "Shoes" category:
- Basketball Shoes: $150, Stock: 8
- Running Socks: $15, Stock: 20
```

**Step 5: Save & Return**
```
Saved to database → Displayed to user with all products
Next time same question asked → Returns cached response instantly
```

## 🔧 Customization

### Change AI Model
Edit [chat/query.py](chat/query.py#L45-L55):
```python
message = client.chat.completions.create(
    model="mixtral-8x7b-32768",  # ← Change model here
    max_tokens=1024,
)
```

Available Groq models:
- `mixtral-8x7b-32768` (default - balanced & fast)
- `llama-2-70b-chat` (more detailed)
- `llama3-70b-8192` (latest & powerful)

### Adjust AI Behavior
Edit system prompt in [chat/query.py](chat/query.py#L28-L36):
```python
system_prompt = f"""You are a helpful product recommendation assistant...
# Modify:
1. Only recommend products that exist in the database  ← Rules
2. Always provide product IDs, names, prices...       ← Requirements
3. Be friendly, concise, and helpful                  ← Tone
"""
```

### Disable Caching
Comment out in [chat/query.py](chat/query.py#L88-L94):
```python
# if history_results and len(history_results) > 0:
#     return cached_response...
```

## 🆘 Troubleshooting

| Issue | Solution |
|-------|----------|
| "ModuleNotFoundError: No module named 'groq'" | Run: `pip install groq` |
| "401 Unauthorized" on Groq API | Regenerate API key and update [chat/query.py](chat/query.py#L6) |
| Chat history not saving | Run: `python chat_migration.py` |
| Slow responses | Topics taking too long? Reduce `max_tokens` from 1024 to 256 |
| Empty search results | Products table might be empty. Check: `SELECT * FROM products;` |
